## Важная техническая оговорка

Сейчас фронтенд работает как React SPA через CDN-модули в `frontend/app.js`.

Это означает:

- React в проекте есть;
- demo поднимается в Docker стабильно;
- но текущий runtime не использует полноценный bundling-пайплайн.

В проекте также остался Vite scaffold:

- `/Users/hanq/PycharmProjects/sber1/frontend/src/App.jsx`
- `/Users/hanq/PycharmProjects/sber1/frontend/src/main.jsx`
- `/Users/hanq/PycharmProjects/sber1/frontend/src/styles.css`
- `/Users/hanq/PycharmProjects/sber1/frontend/package.json`

На данный момент он не используется контейнером и фактически является заделом под следующую итерацию.
